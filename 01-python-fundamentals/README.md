## Materi yang dipelajari:
1. Basic Syntax
2. Data Types
3. Branching dan Looping
4. Functions
5. Data Structures
6. File Handling
7. Basic Error Handling

## Tujuan Pembelajaran
- Memahami dasar pemrograman Python
- Mampu menulis kode yang rapih dan mudah dibaca
- Menyiapkan fondasi sebelum masuk ke Data Analysis dan Machine Learning
